This is the API

getBasicInfoById

Returns a single basic information of a customer for its ID.

URL

http://cobisapi.cobiscorp.com/lastest/customer/basic-info/{idType}/{idNumber}

HTTP Method

GET

Response Type

Customer

Parameters

  • Path: idType String

    Type of customer identification

  • Path: idNumber String

    Number of customer identification
